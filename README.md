# Health Insider (Clinic Management System)
## Prerequisite
- PHP 7 and above
- MySQL

## Dependencies
- FPDF

## Guide to install
1. Create new db named "health_insider"
2. import healthinsider.sql
3. Change the mysql credential in database-conf.php and database/database.php
